Компания такси собрала исторические данные о заказах такси в аэропортах. Чтобы привлекать больше водителей в период пиковой нагрузки, нужно спрогнозировать количество заказов такси на следующий час. Проект описывает процесс построения модели для такого предсказания.
Стэк: python, pandas, scikit-learn, statsmodels, catboost
