# Восстановление золота из руды

Необходимо построить модель машинного обучения для промышленной компании, разрабатывающая решения для эффективной работы промышленных предприятий. Модель должна предсказать коэффициент восстановления золота из золотосодержащей руды на основе данных с параметрами добычи и очистки. Это поможет оптимизировать производство, чтобы не запускать предприятие с убыточными характеристиками. В проекте реализовано построение такой модели.

Стек: python, pandas, numpy, scikit-learn
